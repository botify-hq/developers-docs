Simpleflow

Simple Flow is a Python library that make it easier to write distributed programs with Amazon Simple Workflow. It relies on futures to coordinate the execution tasks.

Logtailer

Logtailer is a tool that parallelize "tail -f" on multiple servers so you can easily follow logs on multiple machines at once.

React Classmap

React Classmap lets you reconcile third-party React components with your CSS framework by defining a mapping of additional class names to apply to children DOM components.
